fifa13bot
=========

A library for scripted searching and trading on the FIFA 13 Auction House.

## Getting Started

Prerequisites:
  * node.js >= v0.8.0

To run the example:

    $ git clone git@github.com:genehallman/fifa13bot.git
    $ cd fifa13bot
    $ cp config.example.js config.js
    $ vi config.js # Fill in the values
    $ npm install
    $ npm start
